About the project

About ClipMerge

ClipMerge is a frontend-ready video generation workspace for turning a short prompt into a stock-footage video.

01

What It Does

ClipMerge accepts a video idea, extracts useful search terms, finds matching free stock footage, and prepares a finished MP4 from those clips.

02

How It Works

The planned Flask backend will receive the prompt and duration, generate keywords, collect clips, arrange them into a timeline, and return progress updates to the interface.

03

Free Stock Footage

The project is designed around freely available stock-footage sources, making it easier to assemble quick concept videos, social clips, and visual drafts automatically.

Workflow

From prompt to video

The current frontend includes the controls, validation, theme system, status messages, and progress helpers needed for backend integration.

  1. Write a prompt describing the video.
  2. Choose the target duration in seconds.
  3. Preview the generated video plan.
  4. Download the completed MP4 when processing finishes.
